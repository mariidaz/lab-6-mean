## Запуск
```bash
npm install (один раз за весь час)
npm start
```

## Примітки
- Якщо використовуєте Node 20+, CRA може показувати попередження. Це шаблон, сумісний з `react-scripts@5`.
- Для чистоти перевірки можна видалити `localStorage.auth` у DevTools або виконати в консолі: `localStorage.removeItem('auth')`.
